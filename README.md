## Gamified To-Do List – React & MongoDB Full-Stack Lessons
*In Progress at SENAI*

**Course:** Technical Course in Systems Development
**Subject:** Application Programming
**Instructor:** Nicolas Marques

This repository contains lessons, exercises, and projects developed throughout the course, covering web programming fundamentals (HTML, CSS, JavaScript), React, and back-end development with MongoDB.

The main project is a Gamified To-Do List, designed to make task management engaging by integrating game mechanics, such as points, levels, or rewards, while practicing full-stack development skills.

### The materials are designed to:

- Build strong foundational skills in web development.
- Enable the creation of dynamic, component-based user interfaces using React (JSX, state management, component lifecycle).
- Explore server-side development using Node.js and MongoDB, including creating APIs and managing databases.
- Integrate gamification features to improve user experience and engagement.
- Running the Front-End (React)

### Install dependencies and start the development server:
To run the React applications, you need Node.js and npm (or yarn) installed.

```bash
npm install
npm start
```

### Running the Back-End (Node.js + MongoDB)

To run the back-end projects, you need Node.js and a MongoDB instance (local or cloud).

Install dependencies and start the server:

```bash
npm install
npm run server
```

Make sure to configure your MongoDB connection string in the environment variables (e.g., .env file).

### Project Structure
/frontend   - React application for the gamified to-do list

/backend    - Node.js + Express + MongoDB projects

/shared     - Common utilities, helpers, or components

### Key Learning Outcomes
- Component-based UI development with React
- State and props management
- API creation and consumption with Node.js and Express
- Database operations with MongoDB (CRUD)
- Full-stack integration
- Gamification implementation to improve user engagement
